# hexo-theme-QAQ
博客的皮肤预览在这[我的博客](http://blog.hiramgao.me/)

鉴于作者我半残疾的设计水平，基本上看见什么好看就添加到页面上，后续会完善移动端的支持和修复BUG以及增加新的功能。

### 安装

需要安装以下依赖

- hexo-renderer-stylus
- hexo-generator-feed

克隆皮肤

`git clone git@github.com:HiramGao/hexo-theme-QAQ.git themes/QAQ`

修改皮肤配置为`QAQ`
